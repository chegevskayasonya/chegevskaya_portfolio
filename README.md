# Софія Чижевська — Портфоліо

Односторінковий сайт-портфоліо fullstack-розробниці. Темна тема з золотим акцентом, плавні скрол-анімації, повна адаптивність — без фреймворків, чистий HTML/CSS/JS.


---

## Структура сторінки

- **Nav** — фіксована навігація, бургер-меню для мобільних
- **Hero** — ім'я, роль, короткий опис, CTA-кнопки, декоративна "code card"
- **Stats** — короткі цифри: проекти, AI-застосунки, % деплоїв на Vercel, час відповіді
- **Projects** — картки проектів з посиланнями на живі демо (Vercel)
- **Services** — перелік послуг (лендинги, AI-застосунки, fullstack, Telegram-боти, UI/UX, підтримка)
- **Stack** — технологічний стек по категоріях: Frontend / Backend / AI & APIs / DevOps
- **Pricing** — три тарифні плани з прозорими цінами в грн
- **Contact** — Telegram, Email (Gmail compose), GitHub, канал з проектами
- **Footer** — копірайт

---


- Scroll-reveal — блоки плавно з'являються при прокручуванні (IntersectionObserver)
- Мобільне меню — бургер-кнопка з анімацією трансформації в хрестик
- Smooth scroll — плавна навігація по якорях
- Прогрес-бар прокрутки — індикатор зверху сторінки
- Кнопка "нагору" — з'являється після прокрутки понад 500px
- prefers-reduced-motion — анімації автоматично вимикаються для користувачів з відповідними системними налаштуваннями
